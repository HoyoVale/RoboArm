## 串口参数

- 波特率：`9600`
- 数据格式：`8N1`
- 默认端口示例：`COM7`

注意：

- 同一时刻只能有一个程序占用串口
- 正式上位机程序运行时，不要同时开启 `shell` 调试模式
- 推荐上位机程序启动后保持串口常开，不要频繁开关

## 模块导入方式

如果你的主程序放在项目根目录下，可以这样导入：

```python
from tools.robot_arm_controller import RobotArmController
```

## 已封装接口

`RobotArmController` 提供这些方法：

- `move_angles({1: 60, 2: 70, 3: 60}, 800)`
- `move_pulses({1: 1500, 2: 1600}, 800)`
- `pump_on()`
- `pump_off()`
- `valve_close()`
- `valve_open()`
- `grab(500)`
- `release()`
- `stop()`
- `home(800)`
- `save_home()`
- `send_raw("#1A60T800")`

## 关节定义

- `1号舵机`：底座绕 Z 轴旋转
- `2号舵机`：大臂前伸后缩
- `3号舵机`：小臂上下运动

## 当前默认 Home 位姿

当前代码中的默认位姿是：

- `1号 = 60°`
- `2号 = 70°`
- `3号 = 60°`

如果已经执行过 `save_home()`，则重启后优先恢复已保存位姿。

## 气泵与电磁阀逻辑

当前硬件已验证可用的抓取流程是：

1. `valve_close()`
2. `pump_on()`
3. 物体吸住后 `pump_off()`
4. 需要放下时 `valve_open()`

也可以直接使用：

- `grab(500)`：抽气一段时间后自动停泵并保持阀门关闭
- `release()`：打开阀门并释放物体

## 最小示例

```python
from tools.robot_arm_controller import RobotArmController

with RobotArmController(port="COM7", baud=9600) as arm:
    arm.home(800)
    arm.move_angles({1: 60, 2: 70, 3: 60}, 800)
    arm.valve_close()
    arm.pump_on()
    arm.pump_off()
    arm.valve_open()
```

## 建议的上位机调用方式

推荐把机械臂控制封装成你视觉程序中的一个长期存在对象，例如：

```python
arm = RobotArmController(port="COM7", baud=9600)
arm.open()
```

然后在视觉循环里按决策调用：

- `arm.move_angles(...)`
- `arm.grab(...)`
- `arm.release()`

程序退出前再：

```python
arm.close()
```

这样更稳，也更接近正式自动控制程序的结构。
